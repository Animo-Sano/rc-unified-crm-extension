Pre-requisites:
- node16+ for esbuild copy

## How To Use

1. Download and unpack chrome extension pack from Github release
2. Install by: Chrome settings -> Extensions -> Load unpacked -> load unpacked `dist` folder

## Features

- Click to dial
- Dialer
- Active call control
- Inbound call popup
- Call log sync
